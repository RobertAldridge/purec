Code for HAVOC3D library for Fast Computation of Generalized Voronoi Diagrams Using Graphics Hardware by Hoff et al in SIGGRAPH 1999 (http://www.cs.unc.edu/~geom/voronoi). 


CONTENTS
--------
havov3d/ - Source code for the library
havoc3d_tutorial/ - A simple program demonstrating use of the library
havoc3d_examples/ - An example program showing some real time computations of Voronoi diagrams
readme.txt - this readme file
license.txt - lincense and contact information

INSTALL
-------
Just unzip the tar file. Visual C++ 6.0 project files are included. Read readme.txt in each subfolder.
